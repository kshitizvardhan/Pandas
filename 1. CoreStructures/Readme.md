### In Pandas, alomst everything we will do involve one of two data structures: the Series and the DataFrame.

- A Series is a 1-dimensional column of data. You can think of it like a single column in a spreadsheet, or a standard Python list, but with a superpower: it has an index (explicit labels for each row).

- A DataFrame is a 2-dimensional table. It is essentially a collection of multiple Series sitting side-by-side, sharing the same index rows. This is where you will spend the vast majority of your time as an AI/ML engineer, as it perfectly represents tabular datasets.
