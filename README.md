# Pandas
Pandas is the foundational Python library for data manipulation and analysis. For an AI/ML engineer, it is the primary tool used to clean, transform, and explore data before feeding it into machine learning models. Think of it as a highly programmable, incredibly fast spreadsheet inside Python. It reads and writes data structures from different formats i.e. csv, json, zip etc.. There are three paths here:

- *The Core Structures*: We can start by comparing the two main building blocks of Pandas: the Series (1-dimensional) and the DataFrame (2-dimensional), including how to create them from scratch.

- *Loading and Inspecting*: We can jump straight into how to load external data (like a CSV file) into Pandas and use basic methods to quickly understand its shape, data types, and missing values.

- *Selecting and Filtering*: If you already have a basic idea of what a DataFrame is, we can dive right into how to extract specific rows, columns, or data points based on conditions.

## What is Data Analysis ?
It is a process of inspecting, cleansing, transforming, and modelling data with the goal of discovering useful information, informing conclusions, and supporting decision-making.
